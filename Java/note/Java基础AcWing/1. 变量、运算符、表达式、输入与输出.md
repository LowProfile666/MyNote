# 变量、运算符、表达式、输入与输出

## 变量

变量必须先定义，才可以使用。不能重名。
变量定义的方式：

```java
int a = 5;
int b, c = a, d = 10 / 2;
```

内置数据类型：

|  类型   | 字节数 | 举例                   |
| :-----: | :----: | ---------------------- |
|  byte   |   1    |                        |
|  short  |   2    |                        |
|   int   |   4    |                        |
|  long   |   8    |                        |
|  float  |   4    | 有效数字是前 6、7 位   |
| double  |   8    | 有效数字是前 15、16 位 |
| boolean |   1    |                        |
|  char   |   2    |                        |
